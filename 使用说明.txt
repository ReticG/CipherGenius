1. 🚀 快速开始
══════════════════════════════════════════════════════════════════════════

   最简单的启动方式:

   👉 双击 start.bat 文件

   首次运行会自动安装依赖，之后会自动打开浏览器。

──────────────────────────────────────────────────────────────────────────

2. 💻 系统要求
══════════════════════════════════════════════════════════════════════════

   ✓ 操作系统: Windows 10/11 (推荐)
   ✓ Python: 3.8或更高版本
   ✓ 内存: 建议4GB以上
   ✓ 磁盘空间: 约2GB (用于依赖包)
   ✓ 网络: 用于安装依赖和调用AI API

──────────────────────────────────────────────────────────────────────────

3. 📦 首次安装
══════════════════════════════════════════════════════────────────────────

   步骤1: 安装Python
   ─────────────────
   如果您还没有安装Python:

   1. 访问 https://www.python.org/downloads/
   2. 下载Python 3.8或更高版本
   3. 安装时勾选 "Add Python to PATH"


   步骤2: 运行启动脚本
   ──────────────────
   1. 双击 start.bat
   2. 首次运行会自动安装依赖（需要几分钟）
   3. 等待浏览器自动打开

──────────────────────────────────────────────────────────────────────────

4. 🎯 启动应用
══════════════════════════════════════════════════════════════════════════

   方法1: 使用启动脚本 (推荐)
   ───────────────────────────
   双击 start.bat 文件


   方法2: 命令行启动
   ──────────────────
   打开命令提示符，进入项目目录，运行:

   python -m streamlit run web_app.py


   方法3: 使用之前的web服务
   ─────────────────────────
   如果后台服务还在运行，直接访问:
   http://localhost:8502

──────────────────────────────────────────────────────────────────────────

5. 🔑 配置API密钥
══════════════════════════════════════════════════════════════════════════

   CipherGenius需要AI服务API才能生成密码学方案。


   步骤1: 复制配置文件
   ──────────────────
   复制 .env.example 文件，重命名为 .env


   步骤2: 编辑配置文件
   ──────────────────
   用记事本打开 .env 文件，填入您的API密钥:

   # 智谱AI (推荐，国内访问快)
   ZHIPUAI_API_KEY=your_zhipuai_key_here

   # 或OpenAI
   OPENAI_API_KEY=your_openai_key_here

   # 或Anthropic
   ANTHROPIC_API_KEY=your_anthropic_key_here


   步骤3: 获取API密钥
   ──────────────────
   ZhipuAI: https://open.bigmodel.cn/
   OpenAI:  https://platform.openai.com/
   Anthropic: https://console.anthropic.com/


   步骤4: 重启应用
   ──────────────────
   关闭并重新打开应用，配置生效

7. ❓ 常见问题
══════════════════════════════════════════════════════════════════════════

   Q1: 双击start.bat后没有反应?
   ────────────────────────────
   A: 检查Python是否正确安装:
      打开命令提示符，输入: python --version
      如果提示未找到命令，需要重新安装Python并勾选"Add to PATH"


   Q2: 浏览器没有自动打开?
   ─────────────────────
   A: 手动打开浏览器，访问 http://localhost:8502


   Q3: 提示端口被占用?
   ──────────────────
   A: 方法1: 关闭占用8502端口的程序
      方法2: 修改web_app.py中的端口号


   Q4: 提示缺少模块?
   ────────────────
   A: 运行命令: pip install streamlit zhipuai


   Q5: API调用失败?
   ───────────────
   A: 检查:
      - .env文件是否存在且配置正确
      - API密钥是否有效
      - 网络连接是否正常
      - API额度是否充足


   Q6: 生成的方案不准确?
   ───────────────────
   A: 尝试:
      - 更详细地描述需求
      - 使用快速示例按钮
      - 指定具体的安全级别和参数


   Q7: 如何停止应用?
   ────────────────
   A: 在命令窗口按 Ctrl+C
      或直接关闭命令窗口


   Q8: 能否离线使用?
   ────────────────
   A: 组件库可以离线浏览，但生成方案需要联网调用AI API
